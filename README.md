# Simple Chatbot Web Application

Dans ce projet, je développe une application web très basique d'un chatbot. L'application comporte une interface utilisateur permettant aux utilisateurs d'entrer des messages et affiche les réponses simulées du bot en utilisant l'API de Llama.

## Fonctionnalités

- **Interface Utilisateur**: Une page avec un champ de texte pour l'utilisateur à entrer son message et un bouton pour envoyer.
- **Chatbot Simulé**: Après l'envoi d'un message par l'utilisateur, le chatbot répond après une courte pause en utilisant l'API de Llama.

## Installation

Pour exécuter ce projet, vous avez besoin d'un navigateur Web moderne. Voici les étapes :

1. Clonez ou téléchargez ce dépôt sur votre ordinateur.
2. Ouvrez le fichier `index.html` dans un navigateur Web.
3. Utilisez l'interface pour entrer des messages et voir les réponses simulées du chatbot.

## API de Llama

Ce projet utilise l'API de Llama pour générer les réponses du chatbot. Vous pouvez trouver plus d'informations sur l'API de Llama [ici](https://llama.example.com/api-documentation).

Pour utiliser l'API de Llama, vous aurez besoin d'une clé API. Assurez-vous de remplacer la clé API dans le fichier `script.js` par votre propre clé.
